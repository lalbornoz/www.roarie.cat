On UI elements (from top to bottom; left to right: canvas; palette, brush, tools, menu) <CTRL> <ALT> <LMB> (while dragging) make UI element floatable & move <CTRL> <RMB> (while dragging) resize UI element <CTRL> <ALT> <SHIFT> <LMB> restore UI element as non-floating and at its original position <CTRL> <ALT> <RMB> restore UI element size These keyboard commands work in brush mode (square, circle, cross): [ brush smaller ] brush bigger <CTRL> [ decrease canvas width <CTRL> ] increase canvas width <CTRL> <ALT> [ decrease canvas height and width <CTRL> <ALT> ] increase canvas height and width <CTRL> <SHIFT> [ decrease canvas height <CTRL> <SHIFT> ] increase canvas height On brush: <CTRL> <LMB> erase cell On canvas: <CTRL> <LMB> draw with bg color <SHIFT> <LMB> draw line from last position <ALT> <LMB> fill brush with sampled color <ALT> <SHIFT> <LMB> copy canvas to brush On custom brush: <SHIFT> <LMB> delete stamp On palette: <RMB> set bg color (when drawing with a letter) Underline tool: <SHIFT> <LMB> remove underlined attribute
